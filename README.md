# Python Data Analysis Project

## 📊 Overview
This project follows the workflow of:
1. **Data Acquisition**: Loading CSVs from URLs.
2. **EDA**: Auditing data types ($Dtypes$) and distributions.
3. **Storage**: Using **DB-API** to save data to SQLite.

## 🛠️ Tech Stack
- **Language**: Python 3.x
- **Libraries**: Pandas, NumPy, Scikit-learn
- **Database**: SQLite3

## 📝 Statistical Methods
The analysis utilizes the standard deviation formula:
$$\sigma = \sqrt{\frac{\sum (x_i - \mu)^2}{n-1}}$$

## 🏃 How to Run
1. Clone the repo.
2. Run `python main.py`.
3. Check `autos.db` for the results.
